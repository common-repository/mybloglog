MyBlogLog Wordpress Plugin: Style Guide

Return to the main documentation

This guide will help you find the necessary selectors to implement CSS rules to alter the members pages and sidebar elements as you wish.

Members Listing Page

div#mybloglog-members

  • h2 - Members List title
  • p.mbl-nav - Pagination (previous/next)
  • p.mbl-listuser - Individual user listing

Individual Member Page

div#mybloglog-member

  • p.mbl-nav - Return to members listing
  • h2 - Screen Name
  • p.mbl-icon - MyBlogLog avatar with link to MyBlogLog profile
  • p.mbl-services - Icons with links to user's listed services (Twitter, Digg, etc.)
  • p.mbl-age - User's age
  • p.mbl-location - User's location
  • p.mbl-bio - User's bio
  • h3 - Heading for sites that the user authors
  • p.mbl-sites - Thumbnail and link to user's site (the class center is also applied to this element)
  • p.listuser - Individual user listing

Widgets

li#mybloglog-members.widget_mybloglog_plugin_members

  • h2 - Widget title
  • p - User avatar and screen name

li#mybloglog-readers.widget_mybloglog_plugin_readers

  • h2 - Widget title
  • p - User avatar and screen name
